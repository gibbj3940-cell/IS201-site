How it works

Simulator rules

The simulator starts with each team’s built-in offense and defense ratings. It then adjusts for home field, tempo, and chaos. Higher chaos creates more randomness, which makes upsets more likely.

Because this is for a web design assignment, the goal is interactivity and clear presentation rather than real predictive accuracy.
